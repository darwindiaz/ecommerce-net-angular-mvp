# Ecommerce .NET + Angular MVP

MVP de e-commerce construido con .NET, Angular, Clean Architecture, Entity Framework Core y SQLite.

## Objetivo

Construir un sistema funcional en tiempo reducido, manteniendo una estructura defendible para una prueba tecnica:

- Autenticacion con JWT.
- Catalogo de productos.
- Carrito persistido en backend.
- Checkout contra entrega.
- Administracion de productos y ordenes.

## Stack propuesto

- Backend: .NET 10+, ASP.NET Core Web API, Entity Framework Core, SQLite, Swagger.
- Frontend: Angular 20+, Signals, RxJS, Reactive Forms.
- Tests: xUnit en backend y minimo un test de componente en Angular.
